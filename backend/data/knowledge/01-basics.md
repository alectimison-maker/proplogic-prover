# 命题逻辑基础

## 一、什么是命题

**命题**是一个能够判断真假的陈述句。每个命题的真值只能是**真（T）**或**假（F）**，不存在第三种可能。

例如：
- "地球是圆的" —— 这是一个命题（真）
- "2 + 3 = 6" —— 这是一个命题（假）
- "请关门" —— 这**不是**命题（祈使句，无法判断真假）
- "x > 5" —— 这**不是**命题（含自由变量，真假不确定）

## 二、命题变量

在命题逻辑中，我们用**大写字母**表示命题变量：

    P, Q, R, S, T, ...

每个命题变量代表一个具体的命题，其值为 T 或 F。

例如：
- 令 P = "今天下雨"
- 令 Q = "我带伞"
- 则"如果今天下雨，那么我带伞"可以形式化为：P → Q

## 三、逻辑联结词

命题逻辑有五个基本联结词，用于将简单命题组合成复合命题：

| 联结词 | 名称     | 读法           | 输入语法 |
|--------|----------|----------------|----------|
| ¬      | 否定     | "非 P"         | ~P       |
| ∧      | 合取     | "P 且 Q"       | P & Q    |
| ∨      | 析取     | "P 或 Q"       | P \| Q   |
| →      | 蕴含     | "如果 P 则 Q"  | P -> Q   |
| ↔      | 等价     | "P 当且仅当 Q" | P <-> Q  |

### 3.1 否定 ¬（NOT）

¬P 表示"P 不成立"。当 P 为真时 ¬P 为假，当 P 为假时 ¬P 为真。

真值表：

    P │ ¬P
    ──┼───
    T │ F
    F │ T

### 3.2 合取 ∧（AND）

P ∧ Q 表示"P 且 Q"。只有 P 和 Q **同时为真**时，P ∧ Q 才为真。

真值表：

    P │ Q │ P ∧ Q
    ──┼───┼──────
    T │ T │  T
    T │ F │  F
    F │ T │  F
    F │ F │  F

### 3.3 析取 ∨（OR）

P ∨ Q 表示"P 或 Q"。P 和 Q **至少一个为真**时，P ∨ Q 为真。
注意：这是**相容析取**（inclusive or），两者都为真时结果也为真。

真值表：

    P │ Q │ P ∨ Q
    ──┼───┼──────
    T │ T │  T
    T │ F │  T
    F │ T │  T
    F │ F │  F

### 3.4 蕴含 →（IF...THEN）

P → Q 表示"如果 P 则 Q"。只有当 P 为真而 Q 为假时，P → Q 才为假。

真值表：

    P │ Q │ P → Q
    ──┼───┼──────
    T │ T │  T
    T │ F │  F
    F │ T │  T
    F │ F │  T

重要说明：当前提 P 为假时，无论 Q 取何值，P → Q 都为真。这叫做**空真**（vacuous truth）。

### 3.5 等价 ↔（IF AND ONLY IF）

P ↔ Q 表示"P 当且仅当 Q"。当 P 和 Q 真值相同时为真。

真值表：

    P │ Q │ P ↔ Q
    ──┼───┼──────
    T │ T │  T
    T │ F │  F
    F │ T │  F
    F │ F │  T

等价关系等同于：(P → Q) ∧ (Q → P)

## 四、合式公式（WFF）

**合式公式**（Well-Formed Formula）是按语法规则正确构造的命题公式：

1. 每个命题变量（P, Q, R, ...）是合式公式
2. 如果 A 是合式公式，则 ¬A 是合式公式
3. 如果 A、B 是合式公式，则 (A ∧ B)、(A ∨ B)、(A → B)、(A ↔ B) 是合式公式
4. 只有通过以上规则构造的才是合式公式

合式公式示例：
- P ∧ Q             ✓
- ¬(P → Q)          ✓
- P → (Q ∨ ¬R)      ✓

非合式公式示例：
- ∧ P Q             ✗（联结词位置错误）
- P ¬ Q             ✗（缺少联结词）

## 五、运算符优先级

为了减少括号的使用，逻辑联结词有如下优先级（从高到低）：

    ¬  >  ∧  >  ∨  >  →  >  ↔

即：¬ 的优先级最高，↔ 最低。

示例：
- ¬P ∧ Q     等价于  (¬P) ∧ Q      （¬ 先于 ∧）
- P ∧ Q ∨ R  等价于  (P ∧ Q) ∨ R   （∧ 先于 ∨）
- P ∨ Q → R  等价于  (P ∨ Q) → R   （∨ 先于 →）
- P → Q ↔ R  等价于  (P → Q) ↔ R   （→ 先于 ↔）

注意：→ 是**右结合**的：
- P → Q → R  等价于  P → (Q → R)

## 六、输入语法指南

在本系统中，输入公式时使用如下替代符号：

| 逻辑符号 | 输入方式 | 示例             | 解释           |
|----------|----------|------------------|----------------|
| ¬        | ~        | ~P               | 非 P           |
| ∧        | &        | P & Q            | P 且 Q         |
| ∨        | \|       | P \| Q           | P 或 Q         |
| →        | ->       | P -> Q           | 如果 P 则 Q    |
| ↔        | <->      | P <-> Q          | P 当且仅当 Q   |
| ⊥        | _|_      | _|_              | 矛盾/假        |

括号使用英文圆括号 ( )，命题变量使用大写英文字母。

输入示例：
- ¬(P ∧ Q) → (¬P ∨ ¬Q)  输入为：  ~(P & Q) -> (~P | ~Q)
- P → (Q → R)           输入为：  P -> (Q -> R)

## 七、重要等价关系

以下等价关系在推理中经常用到：

### 7.1 德摩根定律（De Morgan's Laws）

    ¬(P ∧ Q)  ≡  ¬P ∨ ¬Q
    ¬(P ∨ Q)  ≡  ¬P ∧ ¬Q

"否定合取"等于"析取各自的否定"；"否定析取"等于"合取各自的否定"。

### 7.2 实质蕴含（Material Implication）

    P → Q  ≡  ¬P ∨ Q

蕴含可以改写为析取形式。

### 7.3 双重否定（Double Negation）

    ¬¬P  ≡  P

两次否定等于肯定。

### 7.4 对位律（Contraposition）

    P → Q  ≡  ¬Q → ¬P

一个条件句等价于它的逆否命题。

### 7.5 分配律（Distribution）

    P ∧ (Q ∨ R)  ≡  (P ∧ Q) ∨ (P ∧ R)
    P ∨ (Q ∧ R)  ≡  (P ∨ Q) ∧ (P ∨ R)

### 7.6 吸收律（Absorption）

    P ∧ (P ∨ Q)  ≡  P
    P ∨ (P ∧ Q)  ≡  P

### 7.7 等价的展开

    P ↔ Q  ≡  (P → Q) ∧ (Q → P)

## 八、自然演绎简介

本课程采用**自然演绎**（Natural Deduction）作为证明系统。自然演绎的核心思想是：

- 每个逻辑联结词都有**引入规则**（introduction）和**消除规则**（elimination）
- 引入规则告诉你如何**构造**含有该联结词的公式
- 消除规则告诉你如何**使用**含有该联结词的公式
- 某些规则需要**子证明**（sub-proof）：在假设下推导，然后关闭假设

证明的格式采用 **Fitch 风格**，用竖线和缩进表示子证明的层级：

    1. A          前提
    2. B          前提
    3. | C        假设（子证明开始）
    4. | D        ...推导...
    5. C → D      →-intro 3-4（子证明结束）

后续各章将逐一介绍每个联结词的推理规则。
