# 析取规则（∨）

## 概述

析取（∨）表示"或"的关系。P ∨ Q 为真当且仅当 P 和 Q 至少一个为真。
本章介绍两条规则：**析取引入**（∨-intro）和**析取消除**（∨-elim）。

∨-elim 是一个需要**两个子证明**的规则（分情况讨论），是本课程中结构最复杂的规则之一。

---

## 一、析取引入（∨-intro）

### 形式定义

         A                    B
        ────  ∨-intro        ────  ∨-intro
        A ∨ B                A ∨ B

### 详细说明

如果你已经证明了 A，就可以得到 A ∨ B（对任意 B）。
同理，如果你已经证明了 B，就可以得到 A ∨ B。

直觉理解："A 或 B"只需要一个成立即可。如果我已经知道 A 成立，那么"A 或任何东西"都成立。

### 使用格式

从左分量引入：

    n.  A              （已有）
    k.  A ∨ B          ∨-intro n

从右分量引入：

    n.  B              （已有）
    k.  A ∨ B          ∨-intro n

### 注意事项

- ∨-intro 可以引入**任意**新命题作为析取的另一边
- 你只需要有其中一个分量，另一个分量可以是任何合式公式
- 新引入的分量可以放在左边或右边，取决于目标需要

示例：

    已有：P
    可得：P ∨ Q        （P 在左）
    也可得：Q ∨ P      （P 在右）
    也可得：P ∨ (R → S) （P 在左，右边是任意公式）

## 二、析取消除（∨-elim / 分情况讨论）

### 形式定义

        A ∨ B
        | A           假设（情况 1）
        | ...
        | C
        | B           假设（情况 2）
        | ...
        | C
        ─────────
          C           ∨-elim

### 详细说明

如果你有 A ∨ B（A 或 B 至少一个成立），要从中推出结论 C，需要：

1. **情况 1**：假设 A 成立，推导出 C
2. **情况 2**：假设 B 成立，推导出 C
3. 由于 A ∨ B 保证了 A 和 B 至少一个成立，而无论哪个成立都能得到 C，所以 C 成立

这就是**分情况讨论**（proof by cases）：把所有可能的情况都考虑一遍，每种情况都能得到相同的结论。

### 使用格式

    n.   A ∨ B           （已有）
    n+1. | A              假设（情况 1）
         | ...
    m.   | C              （在情况 1 下得到 C）
    m+1. | B              假设（情况 2）
         | ...
    p.   | C              （在情况 2 下得到 C）
    q.   C                ∨-elim n, (n+1)-m, (m+1)-p

关键要求：
- 两个子证明必须得到**完全相同**的结论 C
- ∨-elim 引用三个部分：析取式的行号，以及两个子证明的范围

## 三、完整示例

### 例题 1（P63）：P ∨ Q ⊢ Q ∨ P

**目标**：证明析取的交换律。

**分析**：
- 有 P ∨ Q，需要得到 Q ∨ P
- P ∨ Q 是析取，使用 ∨-elim 分两种情况
- 情况 1：假设 P，用 ∨-intro 得 Q ∨ P（P 在右）
- 情况 2：假设 Q，用 ∨-intro 得 Q ∨ P（Q 在左）

**证明**：

    1. P ∨ Q             前提
    2. | P               假设（情况 1）
    3. | Q ∨ P           ∨-intro 2
    4. | Q               假设（情况 2）
    5. | Q ∨ P           ∨-intro 4
    6. Q ∨ P             ∨-elim 1, 2-3, 4-5

**步骤解析**：

- **第 1 行**：写出前提 P ∨ Q
- **第 2-3 行**：情况 1 子证明。假设 P 成立，由 P 用 ∨-intro 得到 Q ∨ P（P 是 Q ∨ P 的右分量）
- **第 4-5 行**：情况 2 子证明。假设 Q 成立，由 Q 用 ∨-intro 得到 Q ∨ P（Q 是 Q ∨ P 的左分量）
- **第 6 行**：∨-elim 综合两种情况。前提 P ∨ Q（第 1 行），情况 1（第 2-3 行），情况 2（第 4-5 行），两种情况都得到 Q ∨ P，所以 Q ∨ P 成立

---

### 例题 2：P ∨ Q, P → R, Q → R ⊢ R

**目标**：无论 P 还是 Q 成立，都能得到 R。

**证明**：

    1. P ∨ Q             前提
    2. P → R             前提
    3. Q → R             前提
    4. | P               假设（情况 1）
    5. | R               →-elim 2, 4
    6. | Q               假设（情况 2）
    7. | R               →-elim 3, 6
    8. R                 ∨-elim 1, 4-5, 6-7

**步骤解析**：

- **情况 1**（第 4-5 行）：假设 P，用 →-elim 和前提 P → R 得到 R
- **情况 2**（第 6-7 行）：假设 Q，用 →-elim 和前提 Q → R 得到 R
- **第 8 行**：两种情况都得到 R，所以由 ∨-elim 得到 R

---

### 例题 3：P ∨ P ⊢ P

**目标**：P 或 P，那就是 P（析取幂等律）。

**证明**：

    1. P ∨ P             前提
    2. | P               假设（情况 1）
    3. | P               （就是假设本身）
    4. | P               假设（情况 2）
    5. | P               （就是假设本身）
    6. P                 ∨-elim 1, 2-3, 4-5

两种情况的假设和结论都是 P，直接完成。

---

### 例题 4：(P ∨ Q) ∨ R ⊢ P ∨ (Q ∨ R)

**目标**：析取结合律。需要嵌套使用 ∨-elim。

**证明**：

    1.  (P ∨ Q) ∨ R       前提
    2.  | P ∨ Q            假设（情况 1：左析取成立）
    3.  | | P              假设（情况 1a）
    4.  | | P ∨ (Q ∨ R)    ∨-intro 3
    5.  | | Q              假设（情况 1b）
    6.  | | Q ∨ R          ∨-intro 5
    7.  | | P ∨ (Q ∨ R)    ∨-intro 6
    8.  | P ∨ (Q ∨ R)      ∨-elim 2, 3-4, 5-7
    9.  | R                假设（情况 2：R 成立）
    10. | Q ∨ R            ∨-intro 9
    11. | P ∨ (Q ∨ R)      ∨-intro 10
    12. P ∨ (Q ∨ R)        ∨-elim 1, 2-8, 9-11

## 四、解题策略

### 4.1 使用 ∨-intro 的时机

- 当目标是析取 A ∨ B，且你已经有 A 或 B 时
- 在 ∨-elim 的子证明中，经常需要用 ∨-intro 构造目标

### 4.2 使用 ∨-elim 的时机

- 当前提中有析取式，且你需要从中提取信息时
- 关键判断：是否无法直接确定哪个分量成立，需要分情况讨论

### 4.3 ∨-elim 的思路

1. 确定要对哪个析取式进行分情况讨论
2. 确定两种情况要得到的共同结论 C
3. 分别在每种假设下推导 C
4. 合并两种情况

### 4.4 ∨-intro 添加什么

当需要用 ∨-intro 时，添加的新分量取决于目标：
- 如果目标是 Q ∨ P 而你有 P，就把 P 放右边
- 注意观察目标的结构来决定

## 五、常见错误

### 错误 1：两个子证明结论不一致

    1. P ∨ Q           前提
    2. | P              假设
    3. | R              （某种推导）
    4. | Q              假设
    5. | S              （某种推导）
    6. ???              ∨-elim ✗

∨-elim 要求两个子证明得到**相同的结论**。第一个得到 R，第二个得到 S，无法合并。

### 错误 2：不做分情况讨论就直接使用析取的一个分量

    1. P ∨ Q           前提
    2. P               ✗ ← 不能直接从 P ∨ Q 取出 P！

P ∨ Q 不能保证 P 成立（可能只有 Q 成立）。必须用 ∨-elim 分情况讨论。
（除非你另外有 ¬Q，那可以用 DS。）

### 错误 3：∨-intro 的分量位置搞反

    目标：Q ∨ P
    已有：P
    Q ∨ P    ∨-intro ← 确认 P 在结论中的位置（右边），正确

注意区分 P 在目标中是左分量还是右分量。

## 六、∨-elim 与 DS 的关系

DS（析取三段论）可以看作 ∨-elim 的特殊情况：

    P ∨ Q, ¬P ⊢ Q

用 ∨-elim 证明：

    1. P ∨ Q         前提
    2. ¬P            前提
    3. | P            假设（情况 1）
    4. | ⊥           ¬-elim 3, 2
    5. | Q            ⊥-elim 4（爆炸原理，从矛盾推出任何东西）
    6. | Q            假设（情况 2）
    7. | Q            （就是假设本身）
    8. Q              ∨-elim 1, 3-5, 6-7

DS 将这个过程简化为一步。在本系统中两种方式都可以使用。

## 七、练习

本章内容对应题库中的以下习题：

- **题库第 22-28 题**：析取规则练习
- 这些习题涉及 ∨-intro、∨-elim 以及之前学过的所有规则
- 重点练习 ∨-elim 的分情况讨论结构

**练习策略**：
1. 目标是析取式 → 考虑 ∨-intro，确定已有的是左分量还是右分量
2. 前提中有析取式，且不能直接确定哪边成立 → 使用 ∨-elim
3. ∨-elim 中，先确定两种情况要证明的共同结论
4. 每种情况的子证明中，可以使用所有外层已有的行
5. 复杂题目可能需要在 ∨-elim 的子证明中再使用 →-intro 等其他子证明规则
